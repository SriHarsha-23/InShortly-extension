#  InShortly – Chrome Extension

InShortly is a Chrome extension that summarizes web content into **brief**, **bullet-point**, or **detailed** summaries using your own AI API key.  

---

##  Features
-  **3 Summary Modes**
  - Brief → 3–4 sentence overview  
  - Bullets → 5–10 key points  
  - Detailed → full explanation  

-  **Use Your Own API Key**  
-  **One-Click Summarization**  
-  **Clean & Minimal UI**

---

##  Installation
1. Clone this repo or download the ZIP.  
2. Open Chrome and go to: `chrome://extensions/`  
3. Enable **Developer mode**.  
4. Click **Load unpacked** and select the `InShortly-extension` folder.  
5. Enter your API key in **Settings**, then start summarizing!
