# Privacy Policy for InShortly Extension

InShortly does not collect, store, or share any personal information.  
- The extension only processes the content of the current webpage in order to generate summaries.  
- Any API key entered by the user is stored locally in the browser using Chrome storage and is never shared with third parties.  
- No personal, financial, health, or authentication data is collected.  
- We do not sell or transfer any data to third parties.
